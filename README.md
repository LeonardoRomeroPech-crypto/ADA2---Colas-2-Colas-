Este programa fue desarrollado en Python utilizando la librería Tkinter, la cual permite crear interfaces gráficas.
El objetivo del programa es mostrar cómo funciona una estructura de datos llamada cola y realizar la suma de dos colas de números.

Para el desarrollo del programa utilicé apoyo de herramientas de inteligencia artificial como ChatGPT, que me ayudaron a mejorar el código, modificar el diseño de la interfaz y comprender mejor cómo implementar la estructura de datos.En conclusión, este programa permite comprender de forma visual cómo funcionan las colas en programación, aplicando operaciones como encolar, desencolar y procesamiento de datos mediante una interfaz gráfica.
